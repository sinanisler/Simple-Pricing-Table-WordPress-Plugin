Simple-Pricing-Table-WordPress-Plugin
=====================================

WordPress Simple Pricing Table - WordPress  Fiyat Tablosu Eklentisi. 

**Özellikler**
- Başlık - Title
- Fiyat - Price
- Renk - Color
- Açıklama HTML Kod Girilebilir - Description You can add HTML Codes
- Easy
- Powerfull
